# SABOTAGE

A modern **dark-themed clothing store** built with **React.js**, **Tailwind CSS**, and **Vite**. SABOTAGE is a frontend e-commerce project focused on delivering a clean, responsive, and minimal shopping experience.

> **Note:** This project is currently frontend only. Shopping cart, authentication, search, category filtering, and checkout features are planned for future updates.

---

## ✨ Features

* 👕 Modern clothing store UI
* 🌙 Dark mode interface
* 📱 Fully responsive design
* ⚡ Fast development powered by Vite
* 🔄 Multi-page navigation with React Router
* 🌐 Products fetched from an external API using React Context
* ♻️ Reusable React components
* 🎨 Clean and minimal design

---

## 🛠️ Tech Stack

* React.js
* Tailwind CSS
* Vite
* React Router
* React Context API
* External Product API

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/nabinpoudel2009/Sabotage-React.git
```

### Navigate into the project

```bash
cd Sabotage-React
```

### Install dependencies

```bash
npm install
```

### Start the development server

```bash
npm run dev
```

### Build for production

```bash
npm run build
```
